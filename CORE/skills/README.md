# CORE Skills

Skills are reusable capabilities that blueprints, products, agents, and workflows can compose.

## Examples

- GBP Optimization Skill
- SEO Audit Skill
- Local Competitor Analysis Skill
- AI Visibility Skill
- Review Generation Skill
- Landing Page Skill
- Community Launch Skill
- Revenue Attribution Skill

## Rule

A blueprint is not just a document. A mature blueprint is a collection of skills, prompts, workflows, assets, and proof.

## What Should Not Belong Here

- Client-specific proof.
- Secrets or credentials.
- Unverified tactics promoted as reusable capabilities.
