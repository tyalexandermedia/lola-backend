# TEMPLATES

## Purpose

`TEMPLATES/` is the future home for reusable formats, copy structures, prompts, reports, landing-page sections, onboarding documents, and outreach patterns.

## What Belongs Here

- Reusable prompt templates.
- Report and dashboard copy structures.
- Landing-page section templates.
- Client onboarding templates.
- Email, SMS, and task-list templates.

## What Should Not Belong Here

- Final client-specific copy that belongs under `CLIENTS/`.
- Live production code.
- Secret values or private customer records.
- Shared business logic that belongs in `CORE/`.
- Product strategy that belongs in `PRODUCTS/`.

## How It Supports LOLA OS

`TEMPLATES/` keeps repeatable assets organized so LOLA OS can onboard clients faster while still customizing each client’s final implementation.
