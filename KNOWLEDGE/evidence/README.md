# Evidence Knowledge

Evidence entries will eventually live here or in a backing database.

For now this folder documents the contract: every useful result should be eligible to become structured evidence if it can improve future decisions.

Examples:

- Call converted to estimate.
- Estimate converted to won job.
- Ranking or visibility movement.
- Landing page conversion.
- GBP action.
- Review event.
- Experiment result.
- Revenue attribution event.

Do not store raw private customer data here.
