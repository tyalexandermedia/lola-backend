# Tampa Bay Power Clean

## Purpose

This folder is the future client workspace for Tampa Bay Power Clean, a separate LOLA OS client for Tampa Bay exterior cleaning and pressure washing.

## What Belongs Here

- Tampa Bay Power Clean-specific overview and onboarding notes.
- Market, service-area, and service-line context.
- References to the client dashboard, landing page, revenue workflow, calls, leads, estimates, jobs won, and reporting setup.
- Non-secret integration requirements and verification notes.

## What Should Not Belong Here

- Sandbar configuration, data, docs, or routes.
- Shared platform code that belongs in `CORE/`.
- Generic marketing templates that belong in `TEMPLATES/`.
- Secrets, admin keys, API keys, raw call data, or PII exports.

## How It Supports LOLA OS

This folder keeps Tampa Bay Power Clean separate from Sandbar while giving the client a clear future home for backend, reporting, revenue, and growth-ops context.
