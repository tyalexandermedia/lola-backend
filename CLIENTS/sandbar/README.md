# Sandbar Soft Wash

## Purpose

This folder is the future client workspace for Sandbar Soft Wash, the primary LOLA OS case-study client.

## What Belongs Here

- Sandbar-specific overview and onboarding notes.
- Sandbar market, services, and service-area context.
- References to Sandbar dashboard, reporting, revenue, calls, leads, and ranking workflows.
- Non-secret setup notes for integrations and verification.

## What Should Not Belong Here

- Live Sandbar production code moved from existing modules during Phase 1.
- Secrets, admin keys, API keys, call recordings, or raw PII exports.
- Tampa Bay Power Clean configuration or notes.
- Shared LOLA OS abstractions that belong in `CORE/`.

## How It Supports LOLA OS

This folder creates a dedicated place for Sandbar context without changing the existing Sandbar routes, dashboards, configs, or production behavior.
